1. "b" Tag

The "b" tag is used to make any word in a specific paragraph or any sentence bold, So that everyone got the attention to that specific word.

2. "i" Tag

The "i" tag is used to make a word or sentence inside a paragraph or any sentence a little bit slanted or in a italic form.

3. "u" Tag

The "u" tag os used to make a word or sentence inside a paragraph or any sentence underlined.

4. "select" Tag

The "select" tag is used to creat option based button like structure inside the webpage. By which we can choose any of the following options.

5. "button" Tag

The "button" tag is used to create a button inside the wabepage by which w can navigate to one page to another or to post any details inside a form etc. We can do so many things by the help of that button.

6. "hr" Tag

The "hr" tag id used to create a horizontal line inside the webpage.

7. "br" Tag

The "br" tag id used to break a sentence and add a new line inside our website, In other words we can create a new line inside the website without using repeating the "p" tag again and again.
